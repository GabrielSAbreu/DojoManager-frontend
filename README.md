# DojoManager Frontend

Este projeto é a interface web do sistema Dojo Manager, uma aplicação simples para organizar cadastros de alunos, modalidades e matrículas.

## O que a aplicação faz

A interface permite:
- cadastrar e visualizar usuários;
- cadastrar e visualizar modalidades;
- realizar matrículas de usuários em modalidades;
- editar registros existentes por meio de modais.

## Tecnologias

- HTML
- CSS
- JavaScript

## Como executar ( para testes)

1. Acesse a pasta do projeto.
2. Dê um duplo clique no arquivo index.html

Observação: iniciar o backend disponível em ```https://github.com/GabrielSAbreu/DojoManager-backend```

A aplicação consome os dados de uma API backend em execução localmente.
